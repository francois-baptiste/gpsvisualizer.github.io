http vs. https

If you want to host a GPS Visualizer Google map on a secure connection (i.e., a page whose URL begins with "https://"), you should edit the map so that it calls the Google Maps functions from a site whose URL begins with "https:".

Specifically, replace this:

http://maps.gpsvisualizer.com/google_maps/functions3.js

...with this:

https://gpsvisualizer.github.io/google_maps/functions3.js
